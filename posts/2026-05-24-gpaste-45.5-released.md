It's available [there](https://www.imagination-land.org/files/gpaste/GPaste-45.5.tar.xz), get it while it's hot!

This release targets GNOME 45 to 50.

If you're looking for the legacy applet/tray-icon, it's available [here](https://github.com/Keruspe/gpaste-applet).

## What's new?

- **Bug fix:** clipboard content containing tab characters could trigger an infinite
  Pango assertion loop, eventually filling the disk — this is now fixed (#469)
- Under-the-hood improvements to the daemon and GNOME Shell extension
- Memory management cleanups across the codebase
- Translations updates
- **Note for packagers:** GLib ≥ 2.76 is now required

And don't forget to run `gpaste-client dr` aka `gpaste-client daemon-reexec` after upgrading GPaste to activate new functionalities ;)

You can then run `gpaste-client daemon-version` to check that the correct daemon is running.
